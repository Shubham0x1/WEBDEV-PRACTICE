# ⚖️ BMI Calculator Web Application

A simple and interactive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.  
This project allows users to calculate their BMI based on their **height and weight** and understand their health category.

# 📌 About the Project

The **BMI Calculator** is a web-based application that helps users determine their **Body Mass Index (BMI)** using their height and weight.  

BMI is a commonly used measurement that helps determine whether a person is **underweight, normal weight, overweight, or obese** based on their body mass relative to their height. :contentReference[oaicite:0]{index=0}  

This project demonstrates how **JavaScript can be used to process user input and dynamically display results on a webpage**.

---

# 📊 BMI Formula
BMI = Weight (kg) / (Height (m) × Height (m))

# ⚙️ How It Works

1. The user enters **height and weight** in the input fields.
2. When the **Calculate BMI** button is clicked:
   - JavaScript retrieves the input values.
   - The BMI formula is applied.
3. The calculated BMI value is displayed on the screen.
