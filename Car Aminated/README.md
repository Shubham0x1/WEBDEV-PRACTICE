# 🚗 Car Animation Webpage

A simple and creative **Car Animation Web Project** built using **HTML and CSS**.  
This project demonstrates how animations can be created on a webpage by simulating a **moving car on a road with animated elements**.

# 📌 About the Project

This project showcases a **moving car animation on a webpage** where the car appears to drive along a road while background elements create a sense of motion.  

Animations like these are commonly built using **CSS keyframe animations and positioning techniques** to simulate movement on the screen.

It is a fun beginner-friendly project that helps developers understand **CSS animations, layering, and visual effects**.

# ⚙️ How It Works

1. The webpage loads a **car image and road background**.
2. CSS animations move the road and background to simulate motion.
3. The car wheels rotate using **CSS keyframe animations**.
4. The animation runs continuously in a loop to create a driving effect.
