# 🧮 Calculator Web Application

A simple and interactive **Calculator Web App** built using **HTML, CSS, and JavaScript**.  
This project performs basic arithmetic calculations and demonstrates core concepts of **frontend web development and JavaScript logic building**.

# 📌 About the Project

This project is a **basic calculator** that allows users to perform arithmetic operations such as addition, subtraction, multiplication, and division.  

It is designed as a beginner-friendly project to practice **DOM manipulation, event handling, and UI design in JavaScript**.  

Web calculators built with HTML, CSS, and JavaScript typically include buttons for numbers and operations, a display area for inputs and results, and logic to evaluate expressions dynamically. :contentReference[oaicite:0]{index=0}

# ⚙️ How It Works

1. The calculator interface is created using **HTML buttons and display screen**.
2. **CSS** is used to style the layout and create a clean UI.
3. **JavaScript** listens for button clicks and updates the display.
4. The mathematical expression is evaluated and the result is shown instantly.
