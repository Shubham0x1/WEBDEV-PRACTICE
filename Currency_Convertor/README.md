# 💱 Currency Converter

A simple and interactive **Currency Converter web application** built using **HTML, CSS, and JavaScript**.  
This project allows users to convert one currency into another using real-time exchange rates and provides a clean and user-friendly interface.

# 📌 About the Project

The **Currency Converter** helps users quickly convert values between different currencies.  
It uses JavaScript to fetch exchange rates from an API and dynamically update the converted value when the user changes the currency or amount. Such converters rely on current exchange-rate data to calculate values between currencies. 

This project demonstrates how **JavaScript interacts with APIs and updates the DOM dynamically**.

# ⚙️ How It Works

1. User enters the amount they want to convert.
2. User selects the **source currency** and **target currency**.
3. JavaScript sends a request to an **exchange rate API**.
4. The API returns the latest conversion rate.
5. The converted amount is displayed instantly on the screen.
