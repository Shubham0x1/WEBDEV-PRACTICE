# 🎨 Color Flipper

A simple and interactive **Color Flipper Web Application** built using **HTML, CSS, and JavaScript**.  
This project changes the **background color of the webpage dynamically** when the user clicks a button.

# 📌 About the Project

The **Color Flipper** is a beginner-friendly JavaScript project that generates **random colors** and applies them to the webpage background. Each time the user clicks the button, a new color is generated automatically, creating a dynamic visual effect.  

The project works by generating random RGB or hexadecimal values using JavaScript and updating the background color of the webpage dynamically.

# ⚙️ How It Works

1. The webpage loads with a default background color.
2. The user clicks the **Flip Color** button.
3. JavaScript generates a **random color value**.
4. The background color of the webpage changes instantly.
5. The new color value is displayed on the screen.
