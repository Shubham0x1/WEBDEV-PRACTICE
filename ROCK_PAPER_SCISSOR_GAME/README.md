# 🎮 About the Game

Rock–Paper–Scissors is a classic hand game where two players choose between **rock**, **paper**, or **scissors**.  
Each option defeats one of the others:

- Rock crushes Scissors
- Scissors cut Paper
- Paper covers Rock

The game compares the player's choice with a **random computer choice** and displays the result. :contentReference[oaicite:0]{index=0}

# ⚙️ How the Game Works

1. The player selects **Rock**, **Paper**, or **Scissors**.
2. The computer randomly generates its choice.
3. The game compares both choices.
4. The winner is displayed based on the rules of the game.
5. The score updates automatically.
