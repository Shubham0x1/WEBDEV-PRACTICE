# 📝 To-Do List App

A simple and interactive **To-Do List web application** built using **HTML, CSS, and JavaScript**.  
This project helps users manage their daily tasks efficiently by allowing them to **add, complete, and delete tasks**.

# 📌 Features

✔ Add new tasks  
✔ Mark tasks as completed  
✔ Delete tasks  
✔ Simple and responsive UI  
✔ Interactive user experience using JavaScript

# ⚙️ How It Works

1. User enters a task in the input field.
2. Clicking the **Add button** adds the task to the list.
3. Tasks can be:
   - Marked as **completed**
   - **Deleted** when no longer needed.
4. JavaScript dynamically updates the task list on the page.
