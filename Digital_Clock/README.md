# ⏰ Digital Clock

A simple and interactive **Digital Clock web application** built using **HTML, CSS, and JavaScript**.  
This project displays the **current time in real-time** and updates automatically every second.

👉 https://shubham0x1.github.io/WEBDEV-PRACTICE/Digital_Clock/

# 📌 About the Project

The **Digital Clock** is a beginner-friendly web development project that dynamically displays the current system time using JavaScript. The clock automatically updates every second using JavaScript timing functions and the Date object. :contentReference[oaicite:0]{index=0}

This project is useful for understanding how **JavaScript interacts with the DOM to update content in real time**.

---

# ✨ Features

✔ Displays current time in **hours, minutes, and seconds**  
✔ **Real-time updates every second**  
✔ Simple and clean UI design  
✔ Lightweight and fast  
✔ Beginner-friendly JavaScript project  

---

# 🛠️ Technologies Used

- **HTML5** → Structure of the web page  
- **CSS3** → Styling and layout  
- **JavaScript (Vanilla JS)** → Logic to fetch and update the current time  

JavaScript retrieves the current time using the **Date object** and updates the clock every second using functions like `setTimeout()` or `setInterval()`.

# ⚙️ How It Works

1. JavaScript creates a **Date object** to get the current time.
2. The hours, minutes, and seconds are extracted.
3. The values are displayed on the webpage.
4. The clock updates every second automatically.
