# 🔢 Counter Webpage

A simple and interactive **Counter Web Application** built using **HTML, CSS, and JavaScript**.  
This project allows users to **increase, decrease, and reset a counter value** dynamically using button controls.

# 📌 About the Project

The **Counter Webpage** is a beginner-friendly JavaScript project that demonstrates how web pages can respond to user interactions. The application displays a number (starting from 0) and allows users to modify it using buttons. :contentReference[oaicite:0]{index=0}

# ⚙️ How It Works

1. The counter starts from **0**.
2. When the **Increase** button is clicked, the value increments by 1.
3. When the **Decrease** button is clicked, the value decreases by 1.
4. Clicking **Reset** sets the counter back to **0**.
5. JavaScript updates the displayed value dynamically on the webpage.
